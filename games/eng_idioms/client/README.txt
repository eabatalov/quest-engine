To make plugins avaliable in Construct2:
1. Run cmd.exe as Administrator (right click on cmd.exe file -> run as Administrator)
2. cd "C:\Program Files\Construct 2\exporters\html5\plugins"
3. mklink /D EnglishIdiomsQuestion DEFAULT_STORAGE_GITHUB_DIRECTORY\egp-web-app\games\eng_idioms\client\english_idioms_question_plugin
4. mklink /D EnglishIdioms DEFAULT_STORAGE_GITHUB_DIRECTORY\egp-web-app\games\eng_idioms\client\english_idioms_plugin
